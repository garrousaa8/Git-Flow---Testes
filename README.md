## git clone: clona um repositório já existente na máquina
## git init: inicia um arquivo git na pasta
## git add: prepara arquivos novos ao diretório
## git commit -m: permite fazer um comentário para poder subir o arquivo
## git push: envia o arquivo com as adições e modificações feitas para a nuvem
## git add .: adiciona todos as novas alterações ou arquivos para o repositório
## git pull: baixa as modificações feitas por outras pessoas ou em outras máqui>
## git config --global user.name []: define um nome do criador nos commits
## git config --global user.email []: define o e-mail do criador nos commits
## git status: exibe os arquivos que estão sendo preparados
## git branch: mostra as branchs locais
## git branch [nome]: cria uma nova branch
## git checkout [nome]: alterna entre as branchs
## git merge: mescla a branch específica com a branch atual
